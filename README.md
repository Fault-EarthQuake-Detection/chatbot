# GeoValid Chatbot RAG

Chatbot ini menggunakan model Qwen2.5-1.5B (CPU Mode) dan metode RAG (Retrieval Augmented Generation) untuk menjawab pertanyaan seputar Geologi dan Sesar Patahan.

Dibuat menggunakan:
- FastAPI
- LangChain
- FAISS Vectorstore
- HuggingFace Embeddings
